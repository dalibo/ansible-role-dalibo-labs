Ansible Role : Dalibo Labs Repository
=========

This role installs the Dalibo Labs repo on RHEL/Debian systems.


Example
----------------

```yaml
  - hosts: servers
    roles:
      - dalibo.repo_dalibo_labs
```

Install
----------------

```
$ ansible-galaxy install dalibo.repo_dalibo_labs
```
